# YabadaLang
 Simple interpreter lang for python
## Operators
A limited range of arithmetic expressions are provided. Addition and subtraction have the lowest precedence, but this can be changed with parentheses.
* **+** - Addition
* **-** - Subtraction
* **\*** - Multiplication
* **/** - Division
* **MOD** (or **%**) - Modulo
```
> a = 5+6*2;
> print: a-10;
>> 12
```
